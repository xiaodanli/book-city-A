##### 项目介绍：

小米读书，小米出品的一款Android免费的阅读软件，支持本地与在线阅读，是目前用户口碑最好的阅读软件，不仅具备市面上阅读软件的各种功能，并且具有赏心悦目的全新UI及独创的连载更新等功能。

##### 项目运行

```
git clone git@github.com:xiaodanli/book-city-A.git

npm install --save-dev

npm run dev    //开发环境

npm run build  //线上环境

```
